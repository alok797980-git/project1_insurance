# Insurance Data Analysis Project

## Objective
The objective of this project is to analyze an insurance dataset to understand,
how different factors affect medical insurance charges.
---
##  Dataset
The dataset contains information about individuals including:
- Age
- Gender
- BMI (Body Mass Index)
- Number of children
- Smoking habits
- Region
- Insurance charges (target variable)
---
## Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
---
## Analysis Performed

### 1. Data Cleaning
- Checked for missing values
- Verified dataset consistency

### 2. Univariate Analysis
- Distribution of insurance charges
- Identified skewness in data

### 3. Multivariate Analysis
Analyzed relationship between **charges** and:
- Age
- Gender
- BMI
- Number of children
- Smoking status
- Region

---

##  Key Insights

-  Smokers have significantly higher insurance charges than non-smokers
-  Age has a positive correlation with charges
- Gender has minimal impact on charges
- BMI shows some variation with charges
- Number of children has moderate influence
- Region has little to no impact
---
## Statistical Insight
- Average charges for smokers are much higher compared to non-smokers
---
##  Conclusion
This analysis helps understand the key factors influencing insurance costs, which can assist businesses in pricing strategies and risk assessment.
